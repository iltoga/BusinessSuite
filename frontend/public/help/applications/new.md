# New Application

Create a new visa or service application for a customer.

## Workflow

1. **Select Customer**: Choose an existing customer from the dropdown.
2. **Select Product**: Choose the service/product (e.g., "Visa B211A").
3. **Create**: Click Save to initialize the application and generate the workflow tasks.

Once created, you will be redirected to the **Application Details** view where you can manage the process.
