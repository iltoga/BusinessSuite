# Application Details

Manage the entire lifecycle of a customer application from start to finish.

## 1. Overview & Actions

The header displays the Application ID, Product, and Customer.

- **Create Invoice**: Generate an invoice based on this application.
- **Force Close**: Manually close the application (e.g., if cancelled).
- **Delete**: Remove the application permanently (superuser only).

## 2. Application Info

Manage key metadata card:

- **Dates**: Update the Application Date and Due Date.
- **Calendar**: Check "Add deadlines to calendar" to sync with the system calendar.
- **Notifications**: Enable "Notify customer of deadlines" to send automated alerts via Email/WhatsApp.
- **Notes**: Add internal notes relevant to this application.

## 3. Tasks Timeline (Workflow)

This section tracks the progress of the application through defined steps.

- **Timeline**: Shows past, current, and future tasks.
- **Update Status**: Change the status of the _current_ task (e.g., to "Completed").
- **Auto-Advance**: Completing a task automatically unlocks the next step.
- **Dates**: Adjust Start and Due dates for specific tasks.
- **Rollback**: Use "Cancel Task" to revert a step if a mistake was made.

## 4. Documents

Manage all files related to this application.

### Required & Optional Documents

- **Required**: List of documents that _must_ be uploaded for this product.
- **Optional**: Additional documents that can be added if needed.

### Uploaded Documents

- **Drag & Drop**: Reorder documents in the list.
- **Merge & Download**: Select multiple documents and click "Merge & Download" to get a single PDF.
- **Update/Preview**: View or replace existing files.

### Document Upload Features

- **OCR**: Use "Run OCR" on passports to automatically extract data (Number, Expiration).
- **Expiration**: specific expiration dates for documents.
