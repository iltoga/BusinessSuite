# Products

Manage product catalog entries used to create applications.

## Core actions

- Search by code, name, and description.
- Sort products and open detail records.
- Create, edit, or delete products based on role.
- Export the current catalog to Excel for offline work.
- Print a grouped public price list. The print action generates a retail-only PDF in the background, shows live progress, and then opens the browser/OS print preview automatically.
