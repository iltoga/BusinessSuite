# New / Edit Product

Use this form to create a new product or update an existing one.

## Product Details

- **Name**: Human-friendly product name shown in selectors and lists.
- **Code**: Unique internal identifier used in search, filters, and references.
- **Product type**: Selects visa vs other behavior and labels.
- **Currency**: 2–3 letter code (for example IDR, USD, EUR).
- **Base price**: Internal/base amount.
- **Retail price**: Customer-facing amount (must be >= base price).
- **Validity (days)**: Optional validity duration for the product.
- **Documents min validity**: Minimum remaining validity required for submitted documents.
- **Application window (days)**: Days before expiry of Stay Permit when submission/renewal is allowed.
- **Description**: Internal notes about this product.
- **AI validation prompt**: Optional product-specific instructions used during AI document validation.

## Documents

- **Required documents**: Mandatory documents to complete an application.
- **Optional documents**: Useful but non-mandatory documents.

## Tasks

Define up to 10 workflow tasks.

Each task supports step ordering, duration, optional cost, calendar scheduling, customer reminders, and one **Last step** marker.
